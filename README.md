An approximate 3D model of the main building at UTFPR Ponta Grossa.

The model was created using a Python script running in Blender: 
the entire building was generated via code. 

The reason for this was impatience with Blender's terrible interface,
and the ease of modifying dimensions and the positioning of elements 
— such as windows — directly within the code.

O código pode ser usado assim:
  ./blender --background --python utfpr.py

Executado em Blender 4.5.

A versão original do script foi feita em Blender 2.4.
